Step by Step of How to use this application.



Signing Up

  1. From landing page press ‘sign-up’ shown on the top right.
  2. From the redirected ‘sign-up’ page, input your correct email address, name, password that you would desire to use, then input the same password you used in the ‘confirm password’ section.
  3. Press ‘SIGN UP’ at the bottom of the form.
  4. If incorrect an error message pops up, fix what the message has displayed.
  5. If successful, you are now at the dashboard, from here you may change your details, access your history or add a note.


Login In

  1. From landing page press ‘login’ shown on the top right.
  2. From the ‘login’ page, input the correct email address and password that you used to sign-up with.
  3. Press ‘SUBMIT’ at the bottom of the form.
  4. If incorrect an error message pops up, fix what the message has displayed.


Change Password Process

  1. From Login page, select ‘Forgot Password?’
  2. Input the email to send the forgot password email to.
  3. Go to inputted email address and click on the link provided.
  4. Enter the Email linked to the account the new password and confirm the new password.
  5. Press ‘SUBMIT’.
  6. Login using the new password


Changing Your Details

  1. From Dashboard, go to ‘Change Account Details’ located in navigation bar.
  2. Select ‘Change Details’.
  3. Update the email and name to what you wish.
  4. Press ‘SUBMIT’.

*Note password cannot be changed through this method, for security reasons.



Adding Personal Information

  1. On the Dashboard, enter the subject and body of the note.
  2. Press ‘SUBMIT’.
  3. To view past personal notes, go to history and scroll to find from newest to oldest dates.
